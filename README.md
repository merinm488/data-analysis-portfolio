# Data Analysis Portfolio

A collection of exploratory data analysis projects covering healthcare, sports, and entertainment datasets using Python, Pandas, Matplotlib, and Seaborn.

## Projects

### [Diabetes Patient Data Analysis](./diabetes-analysis/)
Exploratory data analysis on the Pima Indians Diabetes Database. Covers data cleaning, missing value imputation, outlier detection using IQR, correlation analysis, and feature engineering (BMI tiers, OGTT classification).

### [Tokyo Olympics 2021](./tokyo-olympics-2021/)
Analysis of the Tokyo 2020/2021 Summer Olympics dataset with 11,000+ athletes across 47 disciplines. Includes medal tallies per capita, gender distribution across sports, and coaching statistics.

### [Streaming Show Analysis](./show-analysis/)
Data cleaning and analysis of a streaming platform content dataset. Covers duplicate handling, genre distribution, content type analysis, and temporal trends.

### [Algorithm Implementations](./algorithms/)
Python implementations of:
- **N-Queens Problem** - Backtracking solution using recursion
- **Transposition Cipher** - Rail fence encryption and decryption

## Tools & Libraries
- Python 3
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn
